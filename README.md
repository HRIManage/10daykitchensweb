# 10 Day Kitchens

Next.js website for 10 Day Kitchens, including service pages, portfolio content, contact flows, and local location landing pages.

## Tech Stack

- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Vercel deployment
- **Code-First Architecture (No Headless CMS)**
